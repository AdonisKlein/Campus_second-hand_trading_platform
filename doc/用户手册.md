# 校园二手交易平台用户手册

## 1. 系统入口

打开前端首页：

```text
frontend/index.html
```

或通过静态服务器访问：

```text
http://localhost:5500
```

页面顶部导航包含：

- 首页：点击“校园二手交易”返回商品列表。
- 发布物品：进入商品发布页面。
- 订单管理：查看和处理订单。
- 个人中心：登录、查看和修改个人资料。

## 2. 浏览商品

进入首页后，系统会自动加载当前可售商品。

商品卡片显示：

- 商品图片。
- 分类。
- 标题。
- 价格。
- 简短描述。

点击商品卡片可进入商品详情页。

## 3. 搜索和筛选商品

首页提供关键词和分类筛选。

操作步骤：

1. 在关键词输入框输入商品名称关键字，例如“教材”。
2. 在分类下拉框选择分类，例如“书籍”。
3. 点击搜索按钮。
4. 页面刷新商品列表，显示符合条件的商品。

## 4. 注册账号

进入注册页面：

```text
frontend/register.html
```

注册步骤：

1. 输入用户名。
2. 输入 6 到 12 位字母和数字组合的密码。
3. 输入昵称。
4. 输入邮箱。
5. 点击发送验证码。
6. 查看邮箱或后端测试输出，获取验证码。
7. 输入验证码并提交注册。

注意事项：

- 用户名不能重复。
- 邮箱不能重复。
- 验证码错误或过期时注册失败。

## 5. 登录账号

进入个人中心：

```text
frontend/profile.html
```

登录步骤：

1. 输入用户名。
2. 输入密码。
3. 点击登录。

登录成功后，系统显示个人资料。前端会保存当前用户基本信息，用于发布、留言、下单等操作。

登录安全规则：

- 密码错误会提示剩余尝试次数。
- 连续 3 次登录失败后，账号锁定 10 分钟。
- 锁定期间即使输入正确密码也不能登录。

## 6. 修改个人资料

登录后进入个人中心。

操作步骤：

1. 点击“修改资料”。
2. 修改昵称、手机号或邮箱。
3. 如果修改邮箱，需要先发送邮箱验证码。
4. 输入验证码。
5. 点击保存资料。

保存成功后，页面会显示最新资料。

## 7. 发布商品

进入发布页面：

```text
frontend/publish.html
```

发布前需要先登录。未登录用户进入发布页会被提示并跳转到个人中心。

发布步骤：

1. 输入商品标题。
2. 选择分类：书籍、生活用品、电子产品、其他。
3. 输入价格。
4. 输入图片地址，例如 `assets/images/book.svg`。
5. 输入商品描述。
6. 点击提交发布。

发布成功后跳转回首页。

说明：当前版本使用图片地址，不支持从本地选择文件上传。

## 8. 查看商品详情

在首页点击商品卡片进入详情页。

详情页包含：

- 商品图片。
- 商品标题。
- 分类。
- 价格。
- 描述。
- 创建订单按钮。
- 留言沟通区域。

## 9. 留言沟通

详情页右侧为留言沟通区域。

查看留言：

- 游客和登录用户都可以查看留言。

发送留言：

1. 登录账号。
2. 进入商品详情页。
3. 在留言框输入内容。
4. 点击发送留言。
5. 留言成功后，留言列表自动刷新。

## 10. 创建订单

登录用户可在商品详情页点击“创建订单”。

创建规则：

- 不能购买自己发布的商品。
- 商品已被下单或已售出时不能重复下单。
- 创建成功后商品状态变为 `SOLD`。

## 11. 管理订单

进入订单管理页：

```text
frontend/orders.html
```

页面会显示当前用户作为买家或卖家的订单。

订单信息包括：

- 订单编号。
- 物品名称。
- 价格。
- 买家。
- 卖家。
- 状态。

支持操作：

- 确认：将状态更新为 `CONFIRMED`。
- 完成：将状态更新为 `COMPLETED`。
- 取消：将状态更新为 `CANCELLED`。

## 12. 退出登录

进入个人中心，点击“退出登录”，确认后退出当前账号。退出后需要重新登录才能发布商品、留言或下单。

## 13. 常见提示

| 提示 | 含义 |
|---|---|
| 请先登录后再发布物品 | 当前没有登录，需要进入个人中心登录。 |
| 请先登录后留言 | 留言功能需要登录。 |
| 请先登录后下单 | 创建订单需要登录。 |
| 物品已被下单或售出 | 商品已不可再次创建订单。 |
| 账号已被临时锁定 | 连续输错密码达到 3 次，需要等待 10 分钟。 |
