# 校园二手交易平台安装手册

## 1. 环境要求

- JDK 17
- Maven 3.8+
- MySQL 8+
- 浏览器：Chrome、Edge 或 Firefox

## 2. 数据库安装

登录 MySQL 后执行：

```sql
source database/schema.sql;
source database/seed.sql;
```

## 3. 后端启动

修改 `backend/src/main/resources/application.yml` 中的数据库用户名和密码。

```bash
cd backend
mvn spring-boot:run
```

## 4. 前端访问

直接打开 `frontend/index.html`，或使用 VS Code Live Server 启动静态页面。

## 5. 常见问题

| 问题 | 处理方式 |
| --- | --- |
| 数据库连接失败 | 检查 MySQL 是否启动、账号密码是否正确 |
| 页面没有数据 | 确认后端已启动且接口地址为 `http://localhost:8080/api` |
| 跨域失败 | 检查后端 `CorsConfig` 是否生效 |
