# 校园二手交易平台需求说明书

## 1. 项目目标

面向在校学生提供简易二手交易工具，支持闲置物品发布、浏览查询、留言沟通、订单跟踪和个人信息管理。

## 2. 功能需求

| 模块 | 功能 | 说明 |
| --- | --- | --- |
| 用户模块 | 注册、登录、个人信息编辑 | 密码需为6-12位字母和数字组合 |
| 物品模块 | 发布、列表、详情、分类查询、关键词搜索 | 支持图片地址字段，后续可扩展上传 |
| 沟通模块 | 按物品留言、查看留言 | 简易留言，不实现实时聊天 |
| 订单模块 | 创建订单、查询订单、更新状态 | 订单状态包含 CREATED、CONFIRMED、FINISHED、CANCELLED |

## 3. 非功能需求

- 页面加载时间控制在3秒内。
- 支持10-20人同时在线的课程演示场景。
- 用户密码不明文存储。
- 界面简洁，适合PC端网页访问。
